# PokeNurse
[![license](https://img.shields.io/github/license/mashape/apistatus.svg?maxAge=2592000)]()
This is a project forked off https://github.com/duhminick/PokeNurse with the original UI.
![Main Window](http://i.imgur.com/wsfNrJf.png)


## Features
* List Pokemon
    * Pokedex Number
    * Name
    * CP
    * IV
* Transfer Pokemon
* Evolve Pokemon

## Development
    git clone https://github.com/duhminick/PokeNurse
    cd PokeNurse
    npm install
    npm start

This project follows the [JS Standard Code Style](http://standardjs.com/index.html) for the most part.
